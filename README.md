# PusherThis is a repo for managing machine readable artifacts for the [Pusher APIs](http://pusher.com/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Pusher APIs](http://pusher.com/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://pusher.com/)- [Developer](http://pusher.com/)- [Twitter](https://twitter.com/pusher)- [GitHub](https://github.com/pusher)The [APIs.json](https://github.com/api-evangelist/pusher/blob/master/apis.json) index provides links to all the relevant links for the Pusher APIs, including the OpenAPI and Postman if they are present.
